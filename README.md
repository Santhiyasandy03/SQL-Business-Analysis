# SQL-Business-Analysis
SQL Business Analysis project using PostgreSQL to analyze 10K+ retail sales records and generate business insights through 20+ SQL queries.

## Project Overview

This project analyzes a retail sales dataset containing over 10,000 records using PostgreSQL. The objective is to generate business insights related to sales, profit, customers, products, regions, and categories through SQL queries.

## Tools Used

* PostgreSQL
* pgAdmin 4
* SQL

## Dataset

* Sample Superstore Dataset
* 10,194 records

## Key Analyses

* Total Sales and Profit
* Regional Sales Performance
* Category and Sub-Category Analysis
* Top Products by Sales
* Top Customers by Revenue
* Segment Analysis
* State-wise Sales and Profit Analysis
* Shipping Mode Performance
* Quantity-Based Product Analysis

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* Aggregate Functions
* COUNT
* SUM
* AVG
* LIMIT

## Files

* business_analysis.sql
* SampleSuperstore.csv

## Outcome

Developed 20+ SQL queries to answer business questions and generate actionable insights from retail sales data.
